11. React Flickr Gallery

> CHECK ME OUT < > PROJECT FILES <

SKILLS USED:

HTML, CSS, React, React Router, Create-react-app Tool, Third-party API, Data Fetching with React, Axios

PROJECT INFO:

Creating an image gallery using React and the Flickr API. Building the gallery components, writing the CSS and setting up routing.
